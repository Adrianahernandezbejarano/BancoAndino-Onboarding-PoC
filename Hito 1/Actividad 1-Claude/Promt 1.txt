Diseña una solución inteligente para el Banco Andino que automatice la detección de inconsistencias entre formularios digitales y documentos cargados por clientes durante el proceso de onboarding de personas naturales. La solución debe abordar los siguientes aspectos:

Objetivo:
- Mejorar la calidad de la información, la eficiencia operativa y la seguridad del onboarding digital.
- Reducir errores manuales, tiempos de validación y riesgos de cumplimiento (KYC, AML, SARLAFT).

Componentes clave a nivel arquitectónico:
- Capa de Cliente: App de onboarding digital, autenticación segura (Cognito).
- Capa de Entrada: API Gateway para recepción de formularios y documentos.
- Capa de Almacenamiento: Amazon S3 para documentos, DynamoDB para formularios y resultados.
- Capa de Procesamiento Inteligente:
- OCR con Amazon Rekognitio- n o Textract.
- IDP/NLP con Amazon Comprehend o SageMaker.
- Comparador de datos (Lambda).
- Motor de reglas e IA (Lambda + SageMaker).
- Capa de Reportes y Alertas:
- Generador de reportes automáticos (Lambda).
- Notificación mediante SQS/SNS.
- Capa de Revisión Humana: Dashboard para analistas, flujo de revisión manual.
- Capa de Seguridad y Auditoría: IAM, KMS, CloudWatch, AWS Config.
